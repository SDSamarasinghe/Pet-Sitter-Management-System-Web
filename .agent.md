## Authentication System

### Token Management (`lib/auth.ts`)
```typescript
getToken(): string | null           // Get JWT from localStorage
setToken(token: string): void       // Store JWT
removeToken(): void                 // Clear JWT
getUserFromToken(): object | null   // Decode JWT payload
isAuthenticated(): boolean          // Check validity + expiration
getUserRole(): string | null        // Extract role
```

### API Client (`lib/api.ts`)
```typescript
// Axios instance with:
// - Base URL from NEXT_PUBLIC_API_URL
// - Request interceptor: auto-attach Bearer token
// - Response interceptor: 401 → redirect to /login

// Exported auth methods:
authApi.login(email, password)
authApi.forgotPassword(email)
authApi.resetPassword(token, newPassword)
```

### Protected Route Pattern
```tsx
"use client";
import { useEffect } from "react";
import { useRouter } from "next/navigation";
import { isAuthenticated } from "@/lib/auth";

export default function ProtectedPage() {
  const router = useRouter();
  
  useEffect(() => {
    if (!isAuthenticated()) {
      router.push('/login');
    }
  }, []);
  
  // ... page content
}
```

### Page Component Pattern
```tsx
"use client";

import { useState, useEffect } from "react";
import { useRouter } from "next/navigation";
import api from "@/lib/api";
import { isAuthenticated } from "@/lib/auth";
import { Button } from "@/components/ui/button";
import { Card, CardContent, CardHeader, CardTitle } from "@/components/ui/card";

export default function PageName() {
  const router = useRouter();
  const [data, setData] = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    if (!isAuthenticated()) {
      router.push('/login');
      return;
    }
    fetchData();
  }, []);

  const fetchData = async () => {
    try {
      const response = await api.get('/endpoint');
      setData(response.data);
    } catch (error) {
      console.error('Error:', error);
    } finally {
      setLoading(false);
    }
  };

  if (loading) return <div>Loading...</div>;

  return (
    <div className="container mx-auto p-6">
      {/* Page content */}
    </div>
  );
}
```

## Coding Standards

### DO:
- Add `"use client"` to components that use hooks, state, or browser APIs
- Use `@/` import aliases for all internal imports
- Use `cn()` from `lib/utils` for conditional className merging
- Use shadcn/ui components - don't create custom UI primitives
- Use Tailwind utility classes - avoid custom CSS unless necessary
- Handle loading and error states in all data-fetching components
- Use `next/navigation` for routing (`useRouter`, `usePathname`)
- Optimize images with Next.js `<Image>` component (domain: `res.cloudinary.com`)
- Use responsive Tailwind breakpoints (sm:, md:, lg:, xl:)
- Follow the existing color palette (navy-blue, sky-blue, neutral-bg)

### DON'T:
- Don't use `window` or `localStorage` without checking `typeof window !== 'undefined'`
- Don't store sensitive data in client-side state beyond JWT
- Don't create new UI primitives when shadcn/ui components exist
- Don't use inline styles - use Tailwind classes
- Don't import from `react-dom` directly in App Router components
- Don't use `pages/` directory - this project uses App Router only
- Don't hardcode API URLs - use `NEXT_PUBLIC_API_URL` env var

## Common Tasks

### Adding a New Page
1. Create `app/<route>/page.tsx`
2. Add `"use client"` if it uses hooks/state
3. Add auth check if protected
4. Add to sidebar navigation in `components/layout/Sidebar.tsx`

### Adding a New UI Component (shadcn/ui)
```bash
npx shadcn-ui@latest add <component-name>
```

### Adding API Integration
1. Import `api` from `@/lib/api`
2. Use in `useEffect` or event handlers
3. Handle loading/error/success states
4. Show toast notifications for user feedback

### Form Pattern
```tsx
const [formData, setFormData] = useState({ field: '' });
const [submitting, setSubmitting] = useState(false);

const handleSubmit = async (e: React.FormEvent) => {
  e.preventDefault();
  setSubmitting(true);
  try {
    await api.post('/endpoint', formData);
    toast({ title: "Success", description: "..." });
  } catch (error) {
    toast({ title: "Error", variant: "destructive", description: "..." });
  } finally {
    setSubmitting(false);
  }
};
```

## Troubleshooting

- **NEXT_PUBLIC_ env vars not updating:** Rebuild the app (env vars are embedded at build time)
- **Hydration errors:** Ensure `"use client"` on components using browser APIs
- **Import path errors:** Verify `@/` aliases in tsconfig.json and next.config.mjs
- **Image optimization errors:** Add domains to `next.config.mjs` images.domains
- **API CORS errors:** Backend allows all origins; check if URL is correct
- **Build failures:** Check for TypeScript errors with `npx tsc --noEmit`
